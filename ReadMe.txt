1. pastikan identitas database yang ada di dalam MySQL sudah sesuai dengan yang ada di dalam "app.module.ts" (cek di dalam field imports, typeormmodule)
2. jalankan command : npm install
3. jalankan command : nest start
4. apabila nomor 3 tidak bisa digunakan. gunakan command berikut : npm run start

note : 
- untuk mengetes api, bisa digunakan dengan mengetes menggunakan aplikasi postman
